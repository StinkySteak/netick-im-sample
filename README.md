# Netick IM Sample

This sample demonstrates on how to use Interest Management (using Area of Interest), with the usage of Sandboxing too.

## Highlights
- AoI (Broad Phase Filtering)
- A small usage of Narrow Phase Filtering
- Sandboxing compatible
